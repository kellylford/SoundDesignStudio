This directory contains WAV file exports of all presets from the Sound Design Studio preset library. These files are provided for demonstration purposes to showcase the capabilities of the application without needing to run it.

## Usage Notes

These WAV files demonstrate the range of sounds that can be created with Sound Design Studio's synthesis engine. Each preset combines multiple layers with various synthesis techniques:

- **Waveform synthesis**: Sine, Square, Sawtooth, Triangle waves
- **Envelope shaping**: Attack, Decay, Sustain, Release (ADSR)
- **Harmonics**: Octave, fifth, and sub-bass harmonics
- **FM synthesis**: Frequency modulation with adjustable ratio and index
- **Noise generation**: White, pink, and brown noise with filtering
- **Effects**: LFO modulation and echo

## Regenerating These Files

If you have Sound Design Studio installed, you can regenerate all these demonstration files by running:

```bash
python export_preset_demos.py
```

Or on Windows, simply double-click:
```
export_presets.bat
```

The script automatically detects all presets in the preset library, so if new presets are added, they will be included in the export.

## Custom Directory

You can specify a custom output directory:

```bash
python export_preset_demos.py my_custom_demos
```
